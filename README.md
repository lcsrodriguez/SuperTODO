# SuperTODO - TODO $\TeX$ extension


## Introduction

LaTeX package for TODO commands

### Available environments

| Level  | Background color | Text color (foreground) | Inline | Block |
|:------:|------------------|-------------------------|--------|-------|
|  INSERT      |  GREEN                |     Black                    |  ✅      |  ✅     |
|  UPDATE      |    YELLOW              |      Black                |  ✅      |  ✅     |
|  DELETE      |       RED           |     Black                    | ✅       |  ✅     |
|  REVIEW      |       BLUE           |    Black                     |  ✅      |  ✅     |
